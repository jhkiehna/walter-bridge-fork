# Walter Bridge

## Deploy Instructions

-   Pull repo
-   `composer install`
-   endsure php extensions sybase and odbc are installed

## Running Locally

-   Pull repo
-   `composer install`
-   `cp .env.example .env`
-   Edit environment variables in .env appropriately; fill in DB_WALTER env vars

## Running Tests

-   `./vendor/bin/phpunit`
